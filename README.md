# who-might-be-robots
Our hackathon submission for /hack.

Getting started
======
To get started

Clone the source

Download [node](https://nodejs.org/en/download/)
Download [bower](https://bower.io/)

`
npm install -g bower;
npm install;
bower install;
npm start;
`
